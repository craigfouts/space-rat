Space Rat - Experiments with Gesture-Based Control
==================================================
A webcam-based utility designed to provide an accessible mode of computer interaction for people with impaired mobility. The user can perform ASL-inspired hand gestures for 3D control in Autodesk Fusion and general mouse control in Microsoft Windows. Created for HackOHI/O 2021, where it won 1st place.
